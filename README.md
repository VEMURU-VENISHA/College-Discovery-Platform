# 🎓 College Discovery Platform
## 📌 Overview
The College Discovery Platform is a full-stack web application that helps students explore colleges based on various criteria like location, courses, and ratings.
## 🚀 Features
* 🔍 Search colleges
* 🎯 Filter by location, course, and rating
* 📄 View detailed college information
* 💾 Data stored in database
* 🌐 Responsive UI
## 🛠️ Tech Stack
* Frontend: React.js
* Backend: Spring Boot / Node.js
* Database: MySQL / PostgreSQL
## 🔗 Live Demo
Frontend: https://college-discovery-platform-nine.vercel.app
Backend: https://college-backend-h9y7.onrender.com
## ⚙️ How to Run
### Frontend
cd frontend
npm install
npm start
### Backend
cd Backend
run server
